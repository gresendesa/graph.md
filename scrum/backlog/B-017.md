## Description

Dedicated command to list all nodes that reference a given URI.
Today this is implicit in `tree --refs`. A dedicated command with flat output
and --json support is more ergonomic for agent pipelines.

## Syntax

```bash
mdgraph backlinks docs/db.md#postgres --root docs/
mdgraph backlinks docs/db.md#postgres --root docs/ --json
```

## Output (default)

```
infra.md#infraestrutura
deploy.md#producao
monitoring.md#banco
```

## Output (--json)

```json
{
  "uri": "docs/db.md#postgres",
  "backlinks": [
    {"uri": "infra.md#infraestrutura", "title": "Infrastructure"},
    {"uri": "deploy.md#producao", "title": "Production Deploy"},
    {"uri": "monitoring.md#banco", "title": "Database Monitoring"}
  ]
}
```

## Acceptance criteria

- [ ] Lists all direct backlinks (incoming edges)
- [ ] --json produces structured output
- [ ] Exit code 1 for unknown URI
- [ ] Tests: node with backlinks, node without backlinks, --json
