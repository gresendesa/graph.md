## Description

Compares the current state of a node against a previous git revision.
Reports structural changes: changed metadata fields, added/removed
dependencies. Designed to feed change-awareness into AI agents.

## Syntax

```bash
mdgraph diff docs/auth.md#oauth --since HEAD~10 --root docs/
mdgraph diff docs/auth.md#oauth --since abc1234 --root docs/
mdgraph diff docs/auth.md#oauth --since HEAD~10 --root docs/ --json
```

## Output (default)

```
Metadata changes:
  owner: security-team -> platform-team

Dependencies added:
  + security.md#mfa

Dependencies removed:
  - security.md#legacy-auth
```

## Output (--json)

```json
{
  "uri": "auth.md#oauth",
  "revision": "HEAD~10",
  "metadata_changes": {"owner": {"from": "security-team", "to": "platform-team"}},
  "dependencies_added": ["security.md#mfa"],
  "dependencies_removed": ["security.md#legacy-auth"]
}
```

## Notes

- Requires git to be available and the repository to be a git repo
- Uses `git show <rev>:<file>` to retrieve historical content
- Gracefully degrades if not in a git repository
