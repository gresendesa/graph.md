## Descricao

Implementar o comando `mdgraph tree <URI>` (spec section 8.2): consulta o grafo em memoria e exibe
a hierarquia visual de dependencias de uma secao no terminal.

## Escopo

- Comando `tree <URI>`:
  - A partir da URI raiz, navegar `outgoing_edges` recursivamente
  - Renderizar arvore visual no terminal (ex: formato similar ao comando `tree` do Unix)
  - Indicar o tipo de aresta (ref vs include) na representacao visual
- Flag `--refs`:
  - Inverter a direcao: exibir os nos que apontam para a URI solicitada (`incoming_edges`)

## Criterios de aceitacao

- [x] `mdgraph tree arquivo.md#secao` exibe arvore com todas as dependencias diretas e transitivas
- [x] Tipo de aresta visivel na saida
- [x] `mdgraph tree arquivo.md#secao --refs` exibe os backlinks
- [x] Ciclos nao causam loop infinito na exibicao
- [x] URI inexistente exibe erro claro e retorna exit code 1
- [x] Testes em `tests/test_cli_tree.py` (6/6)
