## Description

Returns a compact, structured view of a node and its immediate graph neighborhood.
Designed for AI agent consumption: a single operation that answers
"what do I need to understand this node?" without materializing a full document.

## Syntax

```bash
mdgraph context docs/auth.md#oauth --root docs/
mdgraph context docs/auth.md#oauth --root docs/ --depth 2
mdgraph context docs/auth.md#oauth --root docs/ --json
```

## Output (default)

```
root:      auth.md#oauth
title:     OAuth Authentication
owner:     security-team

includes:
  - security.md#jwt
  - users.md#permissions

referenced_by:
  - login.md#web-login
  - api.md#authentication

metadata:
  type: service
  environment: production
```

## Output (--json)

```json
{
  "uri": "auth.md#oauth",
  "metadata": {...},
  "includes": ["security.md#jwt", "users.md#permissions"],
  "referenced_by": ["login.md#web-login", "api.md#authentication"],
  "content": "..."
}
```

## Acceptance criteria

- [ ] Returns node metadata
- [ ] Returns direct outgoing edges (includes)
- [ ] Returns direct incoming edges (referenced_by)
- [ ] --depth N expands N levels of neighborhood
- [ ] --json produces structured output ready for LLM tool calling
- [ ] Tests cover: leaf node, node with both includes and backlinks, depth>1
