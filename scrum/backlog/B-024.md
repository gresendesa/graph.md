## Description

A specialized variant of `compose` designed specifically for generating
LLM prompt context. Unlike `compose` (which produces a beautiful human-readable
document), `context-compose` produces a compact, token-efficient context block:

- Includes the root node
- Includes mandatory dependencies (direct includes)
- Skips decorative or redundant content
- Respects a configurable depth limit
- Always outputs structured JSON

The goal is not a document for humans — it is a context payload for an AI.

## Syntax

```bash
mdgraph context-compose docs/auth.md#oauth --root docs/
mdgraph context-compose docs/auth.md#oauth --root docs/ --depth 2
mdgraph context-compose docs/auth.md#oauth --root docs/ --max-tokens 4000
```

## Output (--json, always)

```json
{
  "root": "auth.md#oauth",
  "nodes": [
    {
      "uri": "auth.md#oauth",
      "title": "OAuth Authentication",
      "metadata": {...},
      "content": "...",
      "depth": 0
    },
    {
      "uri": "security.md#jwt",
      "title": "JWT Tokens",
      "metadata": {...},
      "content": "...",
      "depth": 1
    }
  ],
  "dependencies": ["security.md#jwt", "users.md#permissions"],
  "total_nodes": 3
}
```

## Acceptance criteria

- [ ] Includes root node content
- [ ] Includes direct dependencies (depth 1 by default)
- [ ] --depth N controls expansion
- [ ] Always outputs JSON (no plain text mode)
- [ ] --max-tokens limits approximate token budget (character estimate)
- [ ] Tests: single node, with deps, depth limit, max-tokens truncation
