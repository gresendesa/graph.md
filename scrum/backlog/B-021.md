## Description

Finds and displays all paths between two nodes in the graph.
Allows an AI agent to justify why two concepts are related — tracing the
dependency chain between them.

## Syntax

```bash
mdgraph explain docs/frontend.md#ui docs/auth.md#oauth --root docs/
mdgraph explain docs/frontend.md#ui docs/auth.md#oauth --root docs/ --json
```

## Output (default)

```
Path 1:
  frontend -> api -> auth -> oauth

Path 2:
  frontend -> api -> postgres
```

## Output (--json)

```json
{
  "source": "frontend.md#ui",
  "target": "auth.md#oauth",
  "paths": [
    ["frontend.md#ui", "api.md#api", "auth.md#oauth"],
    ["frontend.md#ui", "api.md#api", "db.md#postgres"]
  ]
}
```

## Acceptance criteria

- [ ] Finds all simple paths (no repeated nodes)
- [ ] Returns empty list if no path exists (not an error)
- [ ] --json produces structured output
- [ ] Handles disconnected graphs gracefully
- [ ] Tests: direct path, indirect path, no path, --json
