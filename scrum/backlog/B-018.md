## Description

Search nodes by YAML metadata fields. Transforms the `yaml` block into a
queryable semantic index. This is not full-text search — it is structured
field matching.

## Syntax

```bash
mdgraph search tag=infra --root docs/
mdgraph search owner=arquitetura --root docs/
mdgraph search type=database --root docs/
mdgraph search type=database --root docs/ --json
```

## Matching rules

- Single field: `field=value` (exact match on YAML scalar)
- Tag membership: if field is a list, checks containment
- Case-insensitive string comparison

## Output (default)

```
infra.md#postgres
infra.md#redis
infra.md#network
```

## Output (--json)

```json
{
  "query": "type=database",
  "results": [
    {"uri": "infra.md#postgres", "title": "PostgreSQL", "metadata": {...}},
    {"uri": "infra.md#redis",    "title": "Redis Cache", "metadata": {...}}
  ]
}
```

## Acceptance criteria

- [ ] Matches scalar field values (exact)
- [ ] Matches list field membership (tag containment)
- [ ] --json produces structured output
- [ ] Zero results returns empty list (not error)
- [ ] Exit code 1 only on invalid query syntax
- [ ] Tests: exact match, tag match, no results, --json
