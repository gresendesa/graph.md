## Descricao

Implementar o entrypoint CLI `mdgraph` e o primeiro comando: `mdgraph get <URI>` (spec section 8.1).

Extrai uma secao isolada com 100% de fidelidade documental, sem passar pela AST reconstruida.

## Escopo

- Configurar entrypoint CLI usando `typer` (ou `click`)
- Comando `get <URI>`:
  - Resolver a URI para o `ParsedSection` correspondente no SectionIndex
  - Abrir o arquivo fonte
  - Fatirar as linhas `source_start_line` ate `source_end_line` da RawSection
  - Escrever no stdout sem modificacao
- Tratar URI nao encontrada com mensagem de erro e exit code != 0

## Criterios de aceitacao

- [x] `mdgraph get arquivo.md#id-secao` imprime o texto exato da secao no stdout
- [x] Espacos, formatacao e comentarios HTML preservados identicamente ao original
- [x] URI inexistente exibe erro claro e retorna exit code 1
- [x] Testes em `tests/test_cli_get.py` comparando output com conteudo esperado das fixtures (10/10)
