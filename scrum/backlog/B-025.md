## Description

AI agents do not parse ASCII tree diagrams. Every CLI command must support
a `--json` flag that produces a well-typed, consistent JSON response.

`compose` and `context-compose` already have `--json`. This item covers
the remaining commands.

## Commands to update

| Command | Current --json | Action |
|---------|---------------|--------|
| `get`   | No  | Add `{"uri": "...", "content": "..."}` |
| `tree`  | No  | Add `{"uri": "...", "nodes": [...], "edges": [...]}` |

All new commands (B-015 to B-024) must include `--json` by design.

## Acceptance criteria

- [ ] `mdgraph get <URI> --json` returns `{"uri": "...", "content": "..."}`
- [ ] `mdgraph tree <URI> --json` returns `{"uri": "...", "tree": [...]}`
- [ ] All new commands (B-015+) implement --json natively
- [ ] --json output is always valid JSON (parseable by `json.loads`)
- [ ] Tests for --json on get and tree
