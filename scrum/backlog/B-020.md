## Description

Returns the full neighborhood of a node — both outgoing and incoming edges —
up to a configurable depth. Unlike `tree` (outgoing only) or `backlinks`
(incoming only), `neighbors` gives a bidirectional view of the graph around
a node.

Equivalent to "expand context" for an AI agent.

## Syntax

```bash
mdgraph neighbors docs/auth.md#oauth --root docs/
mdgraph neighbors docs/auth.md#oauth --root docs/ --depth 2
mdgraph neighbors docs/auth.md#oauth --root docs/ --json
```

## Output (default)

```
oauth
├─ jwt           (includes)
├─ permissions   (includes)
├─ login         (referenced by)
├─ api           (referenced by)
└─ security      (includes)
```

## Output (--json)

```json
{
  "root": "auth.md#oauth",
  "depth": 2,
  "nodes": [
    {"uri": "...", "relation": "includes", "depth": 1},
    {"uri": "...", "relation": "referenced_by", "depth": 1}
  ]
}
```
