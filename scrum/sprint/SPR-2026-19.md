## Anticipated blockers and mitigations

- Blocker: Path traversal vulnerabilities when extracting files from zip template packages.
  - Mitigation: Ensure strict checking that all target files resolve within the target root using `_is_relative_to` or equivalent checks.
- Blocker: Non-deterministic ZIP creation causing differing checksums for identical templates.
  - Mitigation: Walk through source files in sorted order and use a static `ZIP_EPOCH` (1980-01-01) for ZipInfo timestamps.
