## Manual Validation

Manual tests were conducted using mock server responses and the `tests/test_cli_web_schemas_templates.py` test suite. Verified that:
1. `mdb init` correctly rejects remote HTTP template downloads without a `--checksum` flag.
2. `mdb init` correctly verifies downloaded ZIP content matches the expected `--checksum` SHA-256 hex string and fails gracefully on mismatch.
3. Successful template downloads are cached under `.mdb/cache/templates/` and reused on subsequent `mdb init` runs.
4. `mdb validate` successfully fetches and caches JSON/YAML schemas from remote URIs.
5. `--no-cache` successfully bypasses local cached schemas/templates, forcing a clean HTTP download.
